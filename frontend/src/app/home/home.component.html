Zeitvertreib ist ein anfängerfreundlicher Server für das Spiel SCP: Secret Laboratory auf Steam. Unser Ziel ist es, neuen Spieler:innen den Einstieg in die Welt von SCP: Secret Laboratory zu erleichtern und gleichzeitig erfahrenen Spieler:innen ein erweitertes und komfortableres Spielerlebnis zu bieten.

Standort Server in Nürnberg - zentral in Deutschland, nah an Österreich.
Gameplay Vanilla Feeling mit sinnvollen QoL-Plugins - perfekt abgestimmt.
Moderation Unser aktives Team sorgt für Fairness und schnelle Hilfe.
300+ Spieler Täglich kommen neue Spieler auf unseren Discord!
2 Server Ein zweiter Server steht immer als Backup bereit.
4+ Jahre Unser Projekt besteht seit über vier Jahren!
